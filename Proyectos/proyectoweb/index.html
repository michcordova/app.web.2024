Sobre Mí

Mi nombre es Michelle Cordova soy estudiante de la Universidad Tecnológica de Durango, actualmente estudio la carrera de Tecnologías de la Información.

Las tecnologías de desarrollo web que actualmente utilizamos en los proyectos que se desarrollan son: HTML para la maquetación, CSS hojas de estilo para el diseño, JavaScript programación del lado del cliente, PHP como programación del lado del servidor y como Gestor de Base de Datos MariaDB o MySQL.
